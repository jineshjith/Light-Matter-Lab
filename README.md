# Polarisation dependence of Photophoretic Force
Codes required in the air trap using photophoresis work in LML

This repository contains Jupyter notebooks used to analyse
polarization-dependent absorption of carbon microsphere clusters.

## Repository Structure

notebooks/      → analysis notebooks  
results/        → example outputs  
data_example/   → small dataset for testing
